## ✨ Features

- 💰 Real-time **balance display**
- ⚔️ **Kills & deaths tracking**
- 🏷️ Stats rendered **above player nametags**
- ⚡ Lightweight & **fully client-side**
- 🔐 Works with DonutSMP’s `/api` system



---

## 🚀 Getting Started

### 1. Install Requirements
- Minecraft **1.21.4**
- **Fabric Loader**
- **Fabric API**
- **Java 21+**
- **Modmenu**

### 2. Install the Mod
```bash
mods/
└── donutsmp-balance.jar
```

### 3. In-Game Setup
- Join DonutSMP  
- Run:
```
/api
```
- Enter your API key when prompted
